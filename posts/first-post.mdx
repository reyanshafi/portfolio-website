## Introduction to Next.js

Next.js is a React framework that enables server-side rendering, static site generation, and API routes - making it perfect for modern web applications.

## Key Features

- **Server-Side Rendering**: Better SEO and performance
- **File-based Routing**: Intuitive page structure
- **API Routes**: Build backend endpoints within your app
- **CSS Support**: Built-in support for CSS modules
